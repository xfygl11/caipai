# 在线云打包 APK 说明

推荐使用 GitHub Actions 云打包。你只需要手机或电脑浏览器上传项目，GitHub 会自动生成 APK。

## 第一步：上传项目

1. 打开 GitHub。
2. 新建一个仓库，例如 `dlt-android-app`。
3. 上传本项目文件夹里的所有文件。
4. 确认 `.github/workflows/build-apk.yml` 也上传成功。

## 第二步：开始打包

1. 进入 GitHub 仓库页面。
2. 点击 `Actions`。
3. 选择 `Build Android APK`。
4. 点击 `Run workflow`。
5. 等待构建完成。

## 第三步：下载 APK

1. 打开刚完成的构建记录。
2. 在页面下方找到 `Artifacts`。
3. 下载 `dlt-predictor-debug-apk`。
4. 解压后得到 `app-debug.apk`。
5. 把 APK 发到安卓手机安装。

## 手机安装提示

- 安装时如果提示“未知来源”，需要允许浏览器或文件管理器安装未知应用。
- 这是 Debug APK，首次安装可能会有安全提示，属于正常现象。
- APP 需要联网权限，用于点击 `同步开奖` 获取最新开奖结果。
